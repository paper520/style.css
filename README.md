## How to use?
如果你开发的是一个web项目，直接在页面引入即可：

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@hai2007/style">
```

如果你想通过npm方式管理，首先你需要通过命令行安装，就像这样：

```bash
npm install --save @hai2007/style
```

安装好了以后，在需要的地方引入即可：

```js
import '@hai2007/style';
```
